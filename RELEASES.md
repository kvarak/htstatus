# Releases

**Purpose**: User-focused summaries of new features and improvements in HattrickPlanner
**Audience**: Hattrick managers who want to understand what's new without technical details
**Format**: Only versions with user-facing changes that you'll actually notice

*For detailed technical changes, see [CHANGELOG.md](CHANGELOG.md)*

---

## 3.22 - February 5, 2026

- **Comprehensive Match Analytics**: Deep dive into your match performance with detailed statistics
  - **Possession Data**: See ball possession percentages for each half - track your team's control
  - **Chance Breakdown**: View scoring opportunities by position (left, center, right, set pieces, special events)
  - **Positional Ratings**: Detailed ratings for defense and attack across all field positions
  - **Match Environment**: Arena attendance, weather conditions, referee information
  - **Formations & Tactics**: See exactly how both teams lined up and what tactics were used
- **Redesigned Matches Page**: Clean card-based layout with match history sidebar and detailed analytics panel
  - Win/Draw/Loss badges for quick match result scanning
  - Click any match to see full analytics and tactical breakdown
  - Upcoming matches show opponent history to help you prepare
- **Archive Download**: Fetch historical match data from past seasons for long-term analysis
- **Visual Performance Indicators**: Color-coded ratings and gradient bars make it easy to spot strengths and weaknesses at a glance

*Note: Some match features still in development - look for the "Work in Progress" badge*
